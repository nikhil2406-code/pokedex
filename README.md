# Pokédex Web Application

A Flask-based Pokédex web application that displays Pokémon data using the PokeAPI.

## Features

- Browse all Pokémon
- Filter Pokémon by type
- Live search by name or ID
- Detailed Pokémon information page
- Fast rendering with lazy loading
- Responsive card layout

## Technologies Used

- Python
- Flask
- HTML / CSS / JavaScript
- PokeAPI

## Installation

Clone the repository:

git clone https://github.com/nikhil2406-code/pokedex.git

Install dependencies:

pip install -r requirements.txt

Run the application:

python app.py
